# Findings

This folder contains individual finding records for each vulnerability identified during the assessment.

## Filing Convention

Each finding is saved as a separate file using the naming convention:  
`FIND-[NNN]-[short-title].md`

Example: `FIND-001-sql-injection-login.md`
