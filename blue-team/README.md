# Blue Team Projects

This section documents defensive security projects focused on detection engineering, log analysis, incident response, and threat hunting. These projects demonstrate the ability to monitor environments, identify malicious activity, and respond effectively to security incidents.

---

## Projects

### 1. [Windows Sysmon Detection Lab](./windows-sysmon-detection-lab/)

A hands-on detection engineering lab using Windows Sysmon and a SIEM platform to detect common attacker techniques. Covers log configuration, custom detection rules, alert investigation, and incident reporting.

**Key skills:** Sysmon configuration, event log analysis, detection rule writing, MITRE ATT&CK mapping, incident reporting.

---

### 2. [Phishing Analysis & IR Playbook](./phishing-analysis-ir-playbook/)

End-to-end analysis of simulated phishing emails, including header inspection, attachment analysis, IOC extraction, and the development of an incident response playbook for phishing events.

**Key skills:** Email forensics, IOC extraction, VirusTotal/sandbox analysis, playbook development, user awareness guidance.

---

## Why Blue Team?

Blue team skills are foundational to any security operations role. The ability to detect, investigate, and respond to threats — before they become incidents — is one of the most in-demand capabilities in the industry. These projects demonstrate a structured, evidence-based approach to defensive security work.
