# Sample Emails

This folder contains sanitised, simulated phishing email samples used for analysis in this project.

**Important:** All samples in this folder are entirely fictional and were created for educational purposes. They do not contain real malicious content, real personal data, or real infrastructure indicators.

## Files

| File | Type | Description |
|------|------|-------------|
| `sample-01-credential-harvest.eml` | Credential Harvesting | Simulated login page lure |
| `sample-02-malicious-attachment.eml` | Attachment-based | Simulated invoice with attachment |
| `sample-03-bec-pretexting.eml` | BEC / Pretexting | Simulated wire transfer request |

> *Add your .eml sample files to this folder. Ensure all samples are sanitised before committing.*
