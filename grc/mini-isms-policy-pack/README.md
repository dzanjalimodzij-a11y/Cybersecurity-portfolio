# Mini ISMS Policy Pack

## 1. Project Overview

This project presents a set of six core information security policies aligned to ISO/IEC 27001:2022 requirements. Each policy is written for a fictional small-to-medium organisation and demonstrates the ability to produce professional, enforceable security policies suitable for real-world use. The pack also includes a risk treatment plan, control mapping, and an implementation roadmap.

---

## 2. Scenario

Following the risk assessment conducted for Acorn Solutions Ltd, the Managing Director has approved the development of a foundational Information Security Management System (ISMS). This policy pack forms the governance layer of that ISMS, providing staff with clear, enforceable rules and the organisation with documented evidence of security controls for client and regulatory assurance purposes.
