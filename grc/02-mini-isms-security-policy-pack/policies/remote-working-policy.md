# Remote Working Policy

## Purpose

This policy sets rules for working securely away from the BrightBox office.

## Scope

This policy applies to staff and contractors using laptops, phones, cloud services, email, or business data outside the office.

## Policy requirements

- Staff must use company-approved devices for company work where possible.
- Devices must be kept updated and protected with anti-malware.
- Screens must be locked when unattended.
- Public or shared computers must not be used to access company systems.
- Home Wi-Fi should use a strong password and current router settings.
- Public Wi-Fi should be avoided for sensitive work unless secure cloud access or VPN is in use.
- Company files must be stored in approved cloud services.
- Confidential work should not be visible to unauthorised people in public spaces.
- Lost or stolen devices must be reported immediately.

## Roles and responsibilities

| Role | Responsibilities |
|------|------------------|
| Operations Manager | Communicates remote working expectations. |
| IT Support Provider | Supports device security, updates, and remote access controls. |
| Staff | Protect devices and company information when working remotely. |

## Evidence or records

- device inventory
- remote working acknowledgement
- lost device record
- device security checklist

## Review frequency

This policy should be reviewed annually.
