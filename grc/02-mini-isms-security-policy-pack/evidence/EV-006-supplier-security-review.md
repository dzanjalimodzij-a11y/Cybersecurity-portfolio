EV-006

Supplier security review record

[Supplier Security Policy](../policies/supplier-security-policy.md)

Supplier security and third-party access

## Example record details

| Field | Example entry |
|-------|---------------|
| Supplier | Northgate IT Support Ltd |
| Service provided | Managed IT support and Microsoft 365 administration |
| Data involved | Staff accounts, email administration, device support records |
| Security checks completed | MFA confirmed, named admin accounts confirmed, support access reviewed, incident contact confirmed |
| Risks found | Admin access review not previously documented |
| Decision | Continue supplier relationship with quarterly admin access review |
| Owner | Operations Manager |
| Review date | 2026-06-20 |

## Why this evidence is important

Supplier evidence helps show that third-party access is understood and reviewed. This matters where a supplier can access business systems or sensitive information.

## Review or collection frequency

Review critical suppliers annually and after major service or access changes.

## Notes on limitations

This review is a basic small-business supplier check. Higher-risk suppliers may need contract review, assurance reports, or more detailed due diligence.
