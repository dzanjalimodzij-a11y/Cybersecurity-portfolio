# EV-002 - Access Review Record

## Evidence ID

EV-002

## Evidence title

Quarterly access review record

## Related policy or register

[Access Control Policy](../policies/access-control-policy.md)

## Control area

User access review and least privilege

## Example record details

| System | Users reviewed | Changes required | Owner | Status |
|--------|----------------|------------------|-------|--------|
| Microsoft 365 | 15 staff accounts, 2 shared mailboxes | Remove one inactive shared mailbox delegate | Operations Manager | Open |
| Shopify admin | 5 admin users | Downgrade one marketing user to content-only access | Operations Manager | Open |
| Cloud storage | 6 shared folders | Remove external sharing link from supplier folder | IT Support Provider | Complete |
| Admin accounts | 2 privileged accounts | Confirm named ownership and MFA | IT Support Provider | Complete |

## Why this evidence is important

Access reviews help find accounts or permissions that are no longer needed. For a small company, this is a practical way to reduce account misuse, accidental access, and leaver risk.

## Review or collection frequency

Collect quarterly and after leavers, role changes, or major system changes.

## Notes on limitations

The review is only useful if actions are completed. Open actions should be tracked until closed.
