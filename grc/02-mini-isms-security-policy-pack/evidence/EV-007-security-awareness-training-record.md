# EV-007 - Security Awareness Training Record

## Evidence ID

EV-007

## Evidence title

Security awareness completion record

## Related policy or register

[Security Awareness Policy](../policies/security-awareness-policy.md)

## Control area

Staff security awareness

## Example record details

| Staff role | Training topics | Completion status | Follow-up required |
|------------|-----------------|------------------|-------------------|
| Customer Service Adviser | Phishing, incident reporting, data handling | Complete | None |
| Finance Assistant | MFA, invoice fraud, password manager use | Complete | None |
| Warehouse Supervisor | Device security, incident reporting | Complete | None |
| Marketing Assistant | Shopify access, phishing, cloud sharing | In progress | Complete Shopify access module |
| Operations Manager | Incident logging, access reviews, supplier checks | Complete | None |

## Why this evidence is important

Training records show who has received security guidance and where follow-up is needed. They also help link awareness activity to the risks in the risk register.

## Review or collection frequency

Collect during induction and annual refresher training.

## Notes on limitations

Completion does not prove that staff will always behave securely. It should be supported by reminders, reporting routes, and management follow-up.
