# Evidence

This folder can be used to store future evidence screenshots, records, and review outputs linked to the policy pack. Evidence should be lightweight and useful for showing that the policies are being followed.

Examples of evidence include:

- signed policy acknowledgement
- access review records
- backup restore test result
- MFA configuration screenshot
- incident log
- supplier review record
- security awareness completion record
- leaver checklist
- policy exception record
- management review notes

Evidence should be dated, stored consistently, and linked back to the relevant policy or register where possible.
