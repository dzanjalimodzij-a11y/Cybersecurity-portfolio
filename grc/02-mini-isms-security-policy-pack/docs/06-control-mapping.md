# 06 - Control Mapping

This mapping shows how the policy pack supports common security outcomes. The ISO 27001-style and NIST CSF-style references are used as practical influences, not as certification claims.

| Control area | Relevant policy/register | Purpose | Owner | Evidence that could be collected |
|--------------|--------------------------|---------|-------|----------------------------------|
| Security governance | Information Security Policy; Roles and Responsibilities | Set accountability and approval routes. | Director | Approved policy, management review notes |
| ISMS scope | ISMS Scope; Company Context | Define what the security programme covers. | Director | Approved scope statement |
| Asset awareness | Asset Register | Identify key systems, data, and owners. | Operations Manager | Updated asset register |
| Risk management | Risk Register | Record and prioritise security risks. | Operations Manager | Risk register review notes |
| Access control | Access Control Policy | Limit access to business need. | Operations Manager | Joiner/leaver records, access review |
| Authentication | Password and MFA Policy | Reduce account takeover risk. | IT Support Provider | MFA configuration record |
| Device security | Acceptable Use Policy; Remote Working Policy | Protect laptops and business devices. | IT Support Provider | Device build checklist |
| Data handling | Data Classification and Handling Policy | Set rules for storing, sharing, and disposing of data. | Operations Manager | Staff guidance, file permission review |
| Backup and recovery | Backup Policy | Support recovery from ransomware, deletion, and system failure. | IT Support Provider | Backup logs, restore test result |
| Incident response | Incident Response Policy | Provide a route for reporting and handling incidents. | Operations Manager | Incident log, post-incident review |
| Supplier security | Supplier Security Policy | Manage third-party access and data handling. | Operations Manager | Supplier review record |
| Awareness | Security Awareness Policy | Help staff recognise and report common threats. | Operations Manager | Training completion record |
