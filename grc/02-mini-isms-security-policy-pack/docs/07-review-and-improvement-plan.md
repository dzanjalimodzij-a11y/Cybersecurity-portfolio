# 07 - Review and Improvement Plan

## Review frequency

| Item | Frequency | Owner |
|------|-----------|-------|
| Policy pack review | Annual | Operations Manager |
| Risk register review | Quarterly | Operations Manager |
| Access review | Quarterly | Operations Manager and IT Support Provider |
| Backup restore test | Twice yearly | IT Support Provider |
| Supplier review | Annual for critical suppliers | Operations Manager |
| Security awareness review | Annual | Operations Manager |

## Review triggers

Policies and registers should be reviewed sooner if:

- a security incident occurs
- a new system is introduced
- Shopify, Microsoft 365, or cloud storage use changes significantly
- a supplier gains access to company systems or sensitive data
- a staff role changes in a way that affects access
- legal, contractual, or insurance requirements change

## Internal review process

1. The operations manager collects evidence and open issues.
2. The IT support provider provides technical updates.
3. The risk register is updated with new or changed risks.
4. Policy changes are drafted where needed.
5. The director approves material changes.
6. Staff are told about changes that affect their work.

## Metrics to track

- percentage of users with MFA enabled
- number of overdue leaver access removals
- number of open high-rated risks
- backup restore test result and date
- number of reported phishing emails
- number of incidents and near misses
- percentage of staff with completed awareness training
- number of supplier reviews completed on time

## Improvement actions

| Action | Priority | Owner |
|--------|----------|-------|
| Complete quarterly access reviews | High | Operations Manager |
| Confirm MFA across Microsoft 365, Shopify, and finance systems | High | IT Support Provider |
| Test restore from cloud backup | High | IT Support Provider |
| Create a simple incident log | Medium | Operations Manager |
| Review shared mailboxes and reduce shared credentials | Medium | Operations Manager |
| Run annual phishing awareness refresher | Medium | Operations Manager |

## Management review

The director should review the ISMS annually using a short management pack. The pack should include open risks, policy changes, incidents, access review results, backup testing, supplier issues, and improvement actions.
